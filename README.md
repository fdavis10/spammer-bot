## 📋 О проекте

Spammer Bot — десктопное приложение на Python с графическим интерфейсом (Flet) для массовой рассылки сообщений в чаты Telegram. Поддерживает работу с несколькими аккаунтами, прокси, фильтрацию чатов, шаблоны и автоматическое вступление в каналы.

---

## ✨ Возможности

### 👥 Аккаунты
| Функция | Описание |
|---------|----------|
| **До 10 аккаунтов** | Работа с несколькими Telegram-аккаунтами одновременно |
| **Прокси** | SOCKS5, SOCKS4 и HTTP с опциональной авторизацией для каждого аккаунта |
| **Импорт tdata** | Массовый импорт сессий из папок Telegram Desktop |
| **Проверка валидности** | Вступление в тестовую группу → отправка сообщения → выход → пометка «Валидный» |

### 📤 Рассылка
| Функция | Описание |
|---------|----------|
| **Параллельная отправка** | Все аккаунты рассылают сообщения одновременно |
| **Переменные** | `{имя}`, `{номер}`, `{участников}` в тексте подставляются автоматически |
| **Вложения** | Поддержка фото, документов и других файлов |
| **Повторы при ошибках** | До 3 попыток при FloodWait, ConnectionError, timeout |

### 📂 Чаты
| Функция | Описание |
|---------|----------|
| **Импорт ссылок** | Извлечение ссылок t.me/... из Excel (xlsx) |
| **Параллельное вступление** | Все аккаунты вступают в чаты одновременно |
| **Остановка** | Кнопка «Остановить» во время вступления |

### ⚙️ Фильтрация и тестирование
| Функция | Описание |
|---------|----------|
| **Тестовый режим** | Отправка только в «Избранное» или в один указанный чат |
| **Чёрный/белый список** | Фильтрация чатов по названию |
| **Include/Exclude** | Дополнительные фильтры по названию |

### 📊 Статистика и отчёты
| Функция | Описание |
|---------|----------|
| **Оповещения** | Автоматические уведомления при FloodWait, бане, сетевых ошибках |
| **Экспорт** | CSV и Excel с метриками по датам и типам ошибок |

### 🔐 Безопасность
| Функция | Описание |
|---------|----------|
| **Защита паролем** | Доступ к приложению через мастер-пароль (argon2) |
| **«Запомнить меня»** | Удобный вход без повторного ввода пароля |
| **Экспорт/импорт настроек** | Бэкап конфигурации в JSON |

## 🛠 Установка

### Требования
- Python 3.10+
- [API credentials](https://my.telegram.org) (api_id, api_hash)

### Шаги

```bash
# Клонируй репозиторий
git clone <repo-url>
cd spammer-bot

# Создай виртуальное окружение
python -m venv venv
venv\Scripts\activate

# Установи зависимости
pip install -r requirements.txt

# Инициализируй пароль доступа
python manage.py init-auth

# Запусти GUI
python app.py
```

## 🚀 Быстрый старт

### 1. Первый запуск
1. Выполни `python manage.py init-auth` и задай мастер-пароль.
2. Запусти `python app.py` и войди с этим паролем.

### 2. Настройка (Профиль)
- Введи **api_id** и **api_hash** с [my.telegram.org](https://my.telegram.org).
- Добавь аккаунты (телефон, 2FA при необходимости, прокси опционально).

### 3. Сообщения
- Напиши текст рассылки.
- Используй переменные: `{имя}`, `{номер}`, `{участников}`.
- Добавь вложения при необходимости.

### 4. Чаты
- Импортируй ссылки из xlsx или добавь вручную.
- Нажми «Вступить во все» — аккаунты вступят в каналы/группы.

### 5. Рассылка
- Открой вкладку «Сообщения».
- Нажми «Запустить рассылку» и подтверди.

## ⌨️ CLI (manage.py)

| Команда | Описание |
|---------|----------|
| `python manage.py init-auth` | Задать мастер-пароль |
| `python manage.py setup` | Интерактивная настройка (api_id, api_hash, аккаунты, сообщение) |
| `python manage.py accounts` | Список аккаунтов |
| `python manage.py message` | Показать текущее сообщение |
| `python manage.py message --set` | Задать новое сообщение |
| `python manage.py run` | Запустить рассылку из консоли |
| `python manage.py reset-password` | Сбросить мастер-пароль |

## 🔧 Конфигурация

Основной файл: `config.json` в корне проекта.

- **api_id**, **api_hash** — данные Telegram API
- **accounts** — список аккаунтов (телефон, password 2FA, proxy)
- **message** — текст рассылки
- **templates** — сохранённые шаблоны
- **chat_filter** — фильтры по участникам, include/exclude
- **blacklist** / **whitelist** — списки чатов по названию

Сессии хранятся в `%LOCALAPPDATA%/spammer-bot/sessions/`.
